# Task Manager API (FastAPI)

A resume-ready Task Manager backend built with FastAPI + SQLModel + JWT auth.

## Features
- User signup + login (JWT Bearer auth)
- Projects + Tasks (ownership enforced)
- Filtering tasks (done, query, due date, pagination)
- SQLite for local dev
- Alembic migrations
- Pytest tests

## What were used
FastAPI, SQLModel, SQLite, Alembic, Pytest, JWT (python-jose), bcrypt/passlib

## Run locally (Windows)

### Prerequisites
- Python 3.10+
- Node.js LTS

### One-click start
run run_local.bat
